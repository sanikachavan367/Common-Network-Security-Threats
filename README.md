# DoS & DDoS Attacks
Introduction
A Denial of Service (DoS) attack attempts to make a website or server unavailable by overwhelming it with excessive traffic or requests. A Distributed Denial of Service (DDoS) attack is a stronger version, launched from many infected systems (botnet) at the same time.
Types of Attacks
a) Volume-Based Attacks 
Flood bandwidth with massive traffic.
Examples: UDP Flood, ICMP Flood.
